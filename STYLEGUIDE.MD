# Style Guide

## Project: TaskOps---Ticketing-system

## Project Theme
This application is a ticket management and workflow organization platform. It is designed to help teams manage and support requests, assignments and task throughout the duration of a project.

The design will use a dark mode interface to support dashboards, tickets, ticket details, and a Kanban board.

The overall style should feel clean, efficient and easy to read.
This is because users will need a clear interface with solid spacing that is easy on the eyes during long work sprints and long projects.

## Colour Palette
The application will use the following colours throughout the project.

Midnight Navy `#0F172A` ; Used as the main page background
Slate Panel `#1E293B` ; Used for cards/panels, tickets, Kanban columns
Soft White `#F8FAFC` ; Used for the main text and headings
Muted Gray `#94A3B8` ; Used for secondary text and descriptions
Electric Blue `#3B82F6` ; Used for buttons, links, and active states'

## Fonts
The application should use a sans-serif font for readablity.

## Primary Font used
```css
font-family: Arial, Helvetica, sans-serif;
```
